Protótipo funcional

Somente nesta tela ficam disponíveis as ações de importação, exportação, limpeza de lançamentos e restauração da base de demonstração.

Importe um arquivo JSON exportado pelo sistema quando houver necessidade de restauração ou migração local da base.

Fluxo operacional

  1. Manter cadastros de veículos, motoristas, setores e combustíveis.
  2. Lançar abastecimento com veículo vinculado à placa e combustível.
  3. Validar matrícula, autorização, km e valor total automaticamente.
  4. Consultar por filtros e acompanhar agregados.
  5. Gerenciar dados para integração com API e banco.

Estrutura prevista de produção

  • Front-end web responsivo
  • API REST com autenticação
  • Banco PostgreSQL ou MySQL
  • Auditoria de inclusão, edição e exclusão
  • Perfis: administrador, operador, consulta
